Faiss is a library for efficient similarity search and clustering of dense vectors. It handles sets of vectors of any size — from thousands to billions — and supports both exact and approximate nearest neighbor search. Faiss is written in C++ with complete Python/NumPy bindings, and key algorithms are GPU-accelerated via CUDA.

## Why Faiss?

Faiss is designed for production-scale vector search. It provides a range of index types that trade off search speed, accuracy, and memory usage — giving you full control over the performance envelope for your workload.

## Core workflow

<Steps>
  <Step title="Install Faiss">
    Install via conda for CPU or GPU usage:
    ```bash
    conda install -c pytorch -c conda-forge faiss-cpu=1.14.1
    ```
  </Step>
  <Step title="Build an index">
    Create an index, add your vectors, and start searching:
    ```python
    import faiss
    import numpy as np

    d = 128          # vector dimension
    index = faiss.IndexFlatL2(d)
    vectors = np.random.random((10000, d)).astype('float32')
    index.add(vectors)
    ```
  </Step>
  <Step title="Search">
    Query the index to find nearest neighbors:
    ```python
    query = np.random.random((5, d)).astype('float32')
    k = 10           # number of nearest neighbors
    distances, indices = index.search(query, k)
    ```
  </Step>
  <Step title="Scale up">
    Switch to an IVF or HNSW index for faster approximate search, or move to a GPU index for maximum throughput. See the [Guides](/guides/cpu-indexes) for details.
  </Step>
</Steps>
